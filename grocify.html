Case study03 — Grocify

Grocify

Grocery delivery that feels alive.

Grocify is a cross-platform grocery delivery app built with React Native, designed to make the weekly shop feel quick and friendly. From fluid animations to notifications that actually help, the goal was an app that feels good to open — not one you delete after the first order.

What I built

The feature set.

+Cross-platform grocery delivery built with React Native
+Fluid micro-animations using Lottie
+Location & places via Google APIs
+Push notifications for order updates
+Node.js, Express.js & MongoDB backend
+A smooth cart → order experience

The fun stuff

The challenge & the fix.

A delivery app lives or dies on feel — animations that stutter or notifications that nag will get it deleted before the second order.

I leaned on Lottie for animations that stay buttery without bloating the bundle, pulled in Google APIs for location and places, and tuned push notifications to be genuinely useful — order updates, not noise. A Node, Express and MongoDB backend keeps the whole thing running.

Smooth where it counts, quiet where it should be.
